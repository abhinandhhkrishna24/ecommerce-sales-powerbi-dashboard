# E-Commerce Sales & Fulfillment Dashboard | Power BI

## Project Overview

This project is an end-to-end interactive E-Commerce Sales & Fulfillment Dashboard built using Microsoft Power BI.

The goal of this project was to design a business intelligence dashboard that enables stakeholders to monitor overall sales performance, product profitability, customer behavior, sales channel contribution, and fulfillment operations in a single reporting solution.

The dashboard was built using a custom-generated realistic e-commerce dataset containing 10,000+ transactions across 5 years.

This project simulates a real-world e-commerce analytics environment and focuses on KPI tracking, business storytelling, and interactive dashboard design.

---

# Dataset

This dashboard uses a custom-built sample dataset created to replicate real-world e-commerce business operations.

### Dataset characteristics:
- 10,000+ transactional records
- 5 years of sales history
- Multi-table relational data model
- Realistic product names, brands, customer segments, and operational records

### Dataset includes:

- Orders & transactions
- Product master data
- Customer information
- Marketing campaign data
- Shipping & logistics records

### Key business attributes:

- Order Date
- Product Category
- Brand
- Customer Segment
- Sales Channel
- Payment Method
- Quantity Sold
- Unit Price
- Discount %
- Shipping Cost
- Delivery Cost
- Return Cost
- Delivery Status
- Return Status
- Return Reason
- Customer Rating

The dataset was intentionally designed with realistic scenarios such as:
- seasonal sales spikes
- discounts & campaigns
- returns
- delayed deliveries
- multiple payment behaviors
- customer segmentation
- logistics exceptions

# Dashboard Pages

## 1. Sales Performance Overview

This page provides a high-level executive summary of overall business performance.

### Includes:
- Total Revenue
- Net Revenue
- Net Profit
- Profit Margin %
- Total Orders
- Total Quantity Sold

### Analysis:
- Monthly revenue and order trend
- Profit by customer segment
- Profit by product category
- Regional performance
- Payment method contribution

This page helps business users quickly understand overall sales growth and profitability.

---

# 2. Sales Channel & Product Performance

This page focuses on product performance and channel contribution.

### Includes:
- Revenue by sales channel
- Revenue by customer segment
- Product category profitability
- Payment-wise performance
- KPI comparison across channels
- Dynamic KPI selector using field parameters

### Analysis:
- Which sales channel drives the most revenue
- Which customer segment contributes most to sales
- Best-performing product categories
- Revenue comparison by payment type

This page helps analyze customer purchasing behavior and channel effectiveness.

---

# 3. Fulfillment & Returns Dashboard

This page focuses on operational efficiency and post-sales performance.

### Includes:
- Courier partner performance
- Return trend analysis
- Return reason breakdown
- Late delivery analysis
- Delivery efficiency KPIs
- Return cost tracking

### Analysis:
- Return rate monitoring
- Common reasons for product returns
- Late delivery patterns
- Courier performance comparison
- Logistics cost visibility

This page supports operational and fulfillment decision-making.
